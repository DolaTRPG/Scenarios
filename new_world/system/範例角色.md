# 範例角色

以下範例為玩家提交的角色設定。

```
角色：Dola
種族：外星人
職業：間諜
神之恩寵：讓視線範圍內的所有生物遺忘一件特定的事情

優勢：
  HP5: 外星生物 - 對於所有的動物（人以及非人動物）都擁有相同的觀察能力
  HP3: 觀察者 - 可以冷靜地理解現況、不容易陷入驚慌狀態
  HP1: 外星科技 - 對於所有的科技技術都擁有一眼看穿的能力

地球上有著眾多的外星人
他們透過各種方式隱藏進了人類世界
有的外星人長得就像是人類一樣
有的外星人透過寄生的方式生活在某些人類的身體之中
他們過著與一般人類無異的生活

有些外星人是單純地來地球享受不同種類的生活
有些外星人則是背負著更重大的責任：潛伏在地球上，定期回報人類的文明發展
Dola就是其中一名的潛伏者
20年前就挑選了一名剛出生的人類嬰兒進行替換，讓自己在人類世界成長
如今的他，在其他人的眼中看起來就是一名普通的剛畢業的跳級大學生
但是他自己很清楚自己的使命：「監視人類世界的一切，並在重大改變發生時，跟母星進行回報」

身為外星人的他，在潛伏進地球之前就已經經過完整的間諜訓練
以間諜訓練中心第一名畢業的身份，獲得了高度的肯定
在外星政府的協助下，他被訓練出一種可以改變周圍的生命體記憶的能力，作為保護自己的手段

帶有著外星知識與專業能力的他，雖然身體能力仍然受限於人類身軀的極限
但是知識、記憶、反應完全遠勝過同年紀的人類
他在從小到大的學校生活中一向都是以頂尖成績傲視他人
但是他自己也很清楚過於招搖只會讓自己帶來不利
因此一路保持著低調、與他人友好的態度讓自己可以安定地融入生活之中

如今，在人類學校剛跳級畢業的他，看到了這則來自夢想城市的招募記者消息
他找到了一個可以有效率搜集情報而不會引人注意的好方法
```
