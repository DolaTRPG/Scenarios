# 忒蕾雅

# 背景描述

```
忒蕾雅8月13日出生，
是家中獨生女，血型B型，獅子座。
父親格雷夫是美國黑人，自由業，體內流著愛漂泊冒險，遭遇危機是常有的事情，是個大樂天派。
母親奧堤雅是法國白人，是葡萄酒莊千金，個性天真爛漫，追求刺激又浪漫的愛情，然後是個超大樂天派。

1990-忒蕾莎的父親格雷夫與母親奧堤雅於法國相識。
1991(0)-忒蕾莎8月13日於法國波爾多出生。
1992(1)-忒蕾莎的父母把忒蕾莎丟給外公外婆後，就夫妻一起去美國發展了。
1993(2)-父親格雷夫似乎發展一年就因為受騙而賠盡家產。與母親奧堤雅與其一起流亡，兩人失聯。

1994(3)-沒有進幼稚園，外公外婆開始教忒蕾莎農務以及葡萄酒的各種知識。忒蕾莎被流浪狗群攻擊咬傷，自此對狗有恐懼感。

1995(4)-遇上了初戀，忒蕾莎喜歡上了來莊園工作的美國籍年輕交換學生-傑剛，儘管對方只把自己當小妹妹看，但是忒蕾莎總不放過任何能夠粘著對方的時刻。傑剛是一個對小孩很有耐心，正在修習電腦軟體相關科系的學生。為了更了解傑剛，忒蕾莎盡了全力盧外公外婆，總算是鬧到把未來五年的聖誕禮物全部濃縮成了今年的桌上型電腦。

1996(5)-在每天的農務以及學習不能中斷的前提下，忒蕾莎對於電腦的學習像是本來就有相關的天賦一般的，在傑剛的教導之下，忒蕾莎的C語言學的比英文跟德文甚至是法文都還快的多，甚至已經能明白一些簡單軟體的運作過程了。但在大家驚嘆的同時沒有人知道，這是因為這個小女孩單相思的結晶。

1997(6)-對於要上小學這件事情相當的抗拒，原因自然是跟傑剛的相處時間會減少。但很快的，忒蕾莎就知道傑剛要畢業回到美國的事情。這個小女孩嚐到了第一次不算失戀的失戀酸痛。不過，傑剛與她約定，如果之後電腦能學得跟他一樣厲害的話，在美國一定會再見面的，所以忒蕾莎要好好學習。不過在那之後就與傑剛失去聯絡。

2000(9)-忒蕾莎的國小前期的學業成績維持在前中段。數學比班上的任何人都好。不過對文學或者歷史之類的完全不行。第一次吃到竹筍跟蓮藕，當時覺得是兩個世界上最好吃的食物。在一開始的時候因為膚色的關係有被欺負過，但是很快就沒有了。

2003(12)-忒蕾莎的國小後期，就開始參與軟體設計，科學競賽等比賽。科展時的主題聯繫到葡萄無農藥生產與害蟲物理驅散方式當時受到評審老師的誇讚。

2004(13)-前往巴黎就讀國中，住校宿舍離開故鄉。
2005(14)-在電腦公司當打字員的工讀生。
2006(15)-拒絕了電腦公司的正職人員，準備前往美國的留學考試。因為缺乏運動還有規律生活，身材完全走樣，成為戀愛絕緣體。

2007(16)-對於孫女想要跑去美國，外公外婆最一開始是反對的，但是最終還是放棄了。不過因為英語還有一些文學考試沒有達標準，又重考了一年
2008(17)-考上了賓州的私立高中，玩得很嗨，學業有放在一邊的趨勢。
2009(18)-學業管它去死，忒蕾雅跟同伴整天都泡在娛樂場所裡，連通電話都沒有打回去。
2010(19)-傑剛已經死了，在前年的十二月就已經因事故而辭世。知道這個消息之後，忒蕾雅如大夢初醒。自己完全忘記了為了什麼來到美國。不知道重拾書本來不來的及的，忒蕾莎總算從泥沼中爬了起來。外公外婆知道這樣的狀況後表示理解孫女，並鼓勵她。

2011(20)-大學進入了加州理工學院。並潛心鑽研電腦，幾乎一整天都窩在電腦前。
2012(21)-開始深入暗網，會使用駭客技能使用一些比較危險的方式賺錢，在網上以黑香蕉自稱。研究統計軟體以及機械應用。
2013(22)-在農科企業進行工作，同時修習學業。
2014(23)-回到法國故鄉，透過釀酒葡萄種植以及電子技術的應用來進行專題論文的撰寫。
2015(24)-在農村的生活讓忒蕾雅看起來總算人模人樣的了，開始跑馬拉松當運動。
2016(25)-回到美國的農科企業，繼續進行農業培育軟體研究。

2018(27)-對於農業科技技術的要求，來到新世界進行工作。經調查之後，其生父母皆還活著。
```
